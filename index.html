A javascript library to embed any data at any image file.

Introdunction

画像にデータを埋め込み、それを読み出すためのjsライブラリNameco.jsのデモンストレーションを体験していただけます。
グレーの背景の部分は実際に実行しているプログラムの内容を示しています。
実際にご使用される際の参考としてください。また、このページのソースも、資料としてご活用いただけます。

Step6
